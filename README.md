# Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow using Python, MySQL, and Power BI. The objective is to transform raw data into meaningful insights through data cleaning, exploratory data analysis (EDA), SQL analysis, interactive dashboards, and business reporting.

## Dataset

The project uses a structured dataset containing business-related information such as customers, products, sales, and transactions. The dataset was cleaned and analyzed to identify trends, patterns, and key performance metrics.

## Tools & Technologies

* **Python** – Data loading, cleaning, and exploratory data analysis (EDA)
* **Pandas** – Data manipulation
* **NumPy** – Numerical operations
* **Matplotlib & Seaborn** – Data visualization
* **MySQL** – Data storage and SQL analysis
* **Power BI** – Interactive dashboard creation
* **Gamma** – Presentation (PPT) creation
* **Git & GitHub** – Version control and project hosting

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Checked the dataset structure and data types.

### 2. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected inconsistent data.
* Converted columns to appropriate data types.

### 3. Exploratory Data Analysis (EDA)

* Analyzed data distributions.
* Identified trends and patterns.
* Created visualizations for better understanding.
* Generated summary statistics.

### 4. SQL Analysis

* Imported the cleaned dataset into MySQL.
* Wrote SQL queries to answer business questions.
* Used filtering, grouping, sorting, joins, aggregate functions, and subqueries.

### 5. Power BI Dashboard

Created an interactive dashboard featuring:

* Key Performance Indicators (KPIs)
* Sales and revenue analysis
* Customer insights
* Product performance
* Interactive filters and slicers
* Charts and visualizations for decision-making

### 6. Report

Prepared a business report summarizing:

* Data cleaning process
* EDA findings
* SQL insights
* Dashboard observations
* Business recommendations

### 7. Presentation

Created a professional presentation using Gamma to communicate:

* Project overview
* Analysis process
* Dashboard highlights
* Key findings
* Business recommendations

## How to Run

1. Clone this repository.
2. Install the required Python libraries.
3. Open and run the Jupyter Notebook files.
4. Import the cleaned dataset into MySQL and execute the SQL queries.
5. Open the Power BI (.pbix) file to explore the dashboard.
6. Review the project report and presentation for detailed insights.
